# Blog Fact Checker

A Claude Code plugin that verifies blog posts and articles sentence by sentence, producing a comprehensive verification table with source links and accuracy ratings.

## Features

- Processes all sentences (not just factual claims)
- Outputs results as a markdown table in chat
- Provides source links that support or contradict claims
- Handles technical content, statistics, historical facts, and general knowledge
- Verifies citations and quotes within the blog post

## Usage

```
/blog-fact-checker path/to/blog-post.md
```

Or ask naturally: "fact-check this blog post", "verify the claims in my-article.md"

## Installation

```
/plugin install blog-fact-checker@utibes-plugins-marketplace
```
